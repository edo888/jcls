Adding/Editing Sections

Description

Here you can add or edit sections specific to your project and tag them on the map.

Screenshot

Quick Tips

  • You can tag a road by using the polyline tool.

  • You can add a polygon by using the polygon tool.

  • You can add one road and/or one polygon for a section.
